認識 WebSocket

WebSocket 是基於 TCP 的雙向通信協議，只需要一次 Http Request (握手) 便能持續進行數據互傳。 Http, WebSocket, Socket, TCP 的原理網路上有很多資料，因為不是本文重點就不贅述。(本文要教你的是實作)

分享幾個不錯的文章： 比較 HTTP, TCP, UDP, Socket, WebSocket <-- 其實是不同層面的東西，懶叫比雞腿 WebSocket 原理 Polling, Long-Polling, Streaming 原理
