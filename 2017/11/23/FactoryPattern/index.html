【C#】Factory Pattern

November 23 2017

Factory Pattern 原則

  1. Creation logic encapsulation.
  2. Create object using a common interface.

Factory Pattern 主要四個衍伸概念

  1. Simple Factory
  2. Factory Method
  3. Abstract Factory
  4. Reflection Factory

Factroy Method

Simple Factory 簡單方便，看起來也很合理，
但其實已經違反 OCP 的精神：
Class should allow extending, should prohibit modifying.
因為每新增一個商品，我們就要修改 CreateMerchandise :< 而 Factory Method 的目標就是把各個商品獨立成一個 Factory 就像是企業龍頭發包工作給各個子公司一樣。
